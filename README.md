## Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/eurecah-clone.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd eurecah-clone
    ```

3. **Install dependencies:**

    ```bash
    npm install
    ```

    [or]

    ```bash
    yarn install
    ```
